Intégrer un titre Spotify

  1. Sur Spotify, faites un clic droit sur le titre → Partager → Intégrer le titre.
  2. Choisissez la taille compacte.
  3. Copiez uniquement l'URL contenue dans src="…", sans les guillemets.
  4. Collez-la dans le champ « Lien d'intégration Spotify ».
